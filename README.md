# 도서 정보 관리 시스템

온라인으로 도서정보를 확인하고, 원하는 도서를 관리할 수 있는 웹페이지를 구현하기 위해 시작한 프로젝트입니다.

***

## 사용 기술 스택
- Java & JSP
- MySQL
- HTML/CSS/JAVASCRIPT

***
